# E-Commerce Data Analysis Mini-Project

Welcome to my Data Analysis practice repository! This project is an end-to-end data analysis workflow combining the power of **NumPy** and **Pandas** to clean, analyze, and extract insights from a messy e-commerce dataset.

## Skills Showcased:
* Creating and handling Synthetic Data
* Missing Value Imputation (`fillna`, `dropna`)
* Joining and Merging Datasets (`pd.merge`, `pd.concat`)
* Mathematical Transformations (NumPy)
* Feature Engineering (Binning, custom columns)
* Aggregations and Grouping (`groupby`, `pivot_table`)

*Note: This project is structured in levels to gradually increase the complexity of the data analysis.*
